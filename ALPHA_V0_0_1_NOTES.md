# Camiel alpha-v0.0.1

## Scope

This alpha is a small playable intro for very young children.

## Included

- Camiel as a `CharacterBody2D` player with collision and gravity.
- Idle, walk, run, jump, sit, and sleep animations.
- Simple intro level with a grass floor and a small platform.
- Large child-friendly intro text.
- Basic color blocks: rood, geel, blauw.
- Windows export preset.
- Windows alpha build published as a GitHub Release asset.

## Controls

- Left / right arrows or A / D: walk.
- Shift: run.
- Space, W, or up arrow: jump.
- S or down arrow: sit.
- X: sleep.

## Not Yet Included

- Main menu.
- Save system.
- Audio.
- Score or collectible flow.
- Code signing.
