# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| `alpha-v0.0.1` | Yes |

## Reporting A Vulnerability

Please do not open a public issue for security problems.

Use GitHub's private vulnerability reporting or contact the repository maintainers through the Euraika Labs organization.

Helpful information to include:

- A clear summary.
- Steps to reproduce.
- Affected version or commit.
- Any logs or screenshots that help explain the issue.

## Scope

This is currently a small Godot game prototype. Security concerns are most likely to involve:

- Build or release workflow configuration.
- Downloaded build artifacts.
- Repository or GitHub Actions permissions.
