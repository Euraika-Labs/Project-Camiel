Project Camiel is a small Godot 4 intro game for young children.

## Alpha v0.0.1

- Playable intro scene.
- Camiel as a `CharacterBody2D` player with collision and gravity.
- Idle, walk, run, jump, sit, and sleep animations.
- Simple child-friendly level with color blocks.

## Controls

- Left / right arrows or A / D: walk.
- Shift: run.
- Space, W, or up arrow: jump.
- S or down arrow: sit.
- X: sleep.
